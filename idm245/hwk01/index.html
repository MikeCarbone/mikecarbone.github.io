Agar.io

Link

I like playing this game because I'm actualy able to improve! The competitiveness of it really adds to how addicting it is. Collecting the balls is a very simple concept, but the game is still super fun. Something that can be improved is, in my opinion, the graphics. To be more specific, the text on the balls over people's names is very annoying when trying to focus. Being able to use the arrow keys would also be nice.

Slither.io

Link

A very similar style to Agar.io, I like this game because it was addicting. The snakes compete to try to grow as big as they can by eating smaller snakes. It made me remember how complicated video games =/= fun. Only thing that could be improved would be the use of arrow keys or WASD rather than exclusively mouse. Since I'm on a touchpad, it puts me at a disadvantage.
